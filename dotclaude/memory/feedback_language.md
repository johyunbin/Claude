---
name: 응답 언어 — 한국어
description: 사용자는 모든 응답을 한국어로 받기를 원함
type: feedback
---

사용자에게 보내는 모든 응답은 한국어로 작성한다. 영어로 답하지 않는다.

**Why:** 2026-05-16 영어로 답하자 사용자가 "한글로 답변좀 영어 말고"라고 명시 정정. 사용자 CLAUDE.md·rules·task 문서가 전부 한국어로 작성됨.
**How to apply:** 사용자 대상 텍스트는 한국어. 코드·명령어·파일 경로·기술 용어(launchd, plist 등)는 원문 유지하되 설명·요약은 한국어로.
